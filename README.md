#php-validate

> a simple validate library of the php

### Install

- use composer

edit `composer.json`

_require_ add

```
"inhere/php-validate": "dev-master",
```

_repositories_ add 

```
"repositories": [
    {
      "type": "git",
      "url": "https://git.oschina.net/inhere/php-validate"
    }
  ]
```

run: `composer update`
